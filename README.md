# PS Maker

PS Maker is a no-code game maker for creating 2D top-down, story-driven games. This repo serves as the public issue tracker to submit bug reports, feature requests, and give general feedback. PS Maker is not open source software and this repo **does not** contain any source code for PS Maker. 

→ [Report a bug · Request a feature](https://github.com/pixelstories-hq/ps-maker/issues/new/choose)

## Community

The [Pixel Stories Official Discord](https://discord.gg/WTxUC4hEnS) is where creators come together to connect, brainstorm, build teams, share feedback, playtest, and showcase our games. Join us if you'd like to chat and hang around!
